# Blocky 3D Obby

A small Roblox-style 3D obby game built with HTML, CSS, JavaScript, and Three.js.

## How to Run

Open `index.html` in your web browser.

That is all you need. There is no install step because the game is a static website.

Note: the game loads Three.js from a CDN, so you need an internet connection when opening the page.

## Controls

- `W` or `Arrow Up`: move forward
- `S` or `Arrow Down`: move backward
- `A` or `Arrow Left`: move left
- `D` or `Arrow Right`: move right
- `Space`: jump
- Click and drag on the game screen: rotate the camera in normal mode
- `Shift`: toggle Shift Lock, then click the game screen once to lock the mouse
- In Shift Lock, move the mouse to rotate the camera and character without holding the mouse button

On small screens, touch controls appear automatically.

## Settings

Use the **Settings** button in the game to change:

- Camera sensitivity
- Movement speed
- FPS limit
- Graphics quality
- Shadows
- Shift Lock
- Touch controls
- Fullscreen

## Files

- `index.html`: page structure
- `styles.css`: layout and UI styles
- `game.js`: Three.js scene, player movement, settings, and gameplay
